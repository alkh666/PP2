# TSIS2 Paint Application

Extended Paint application using only Pygame built-in drawing tools.

## Run

```bash
pip install pygame
python3 paint.py
```

## Tools

- Pencil: freehand drawing
- Line: straight line with live preview
- Rectangle
- Square
- Circle
- Right triangle
- Equilateral triangle
- Rhombus
- Eraser
- Flood fill
- Text tool

## Controls

- `1` - small brush, 2 px
- `2` - medium brush, 5 px
- `3` - large brush, 10 px
- `Ctrl+S` - save canvas as timestamped PNG into `saves/`
- Text tool: click canvas, type, press `Enter` to confirm, `Esc` to cancel
